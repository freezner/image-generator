요구사항 정의서 (PRD): 브랜드 일관성 유지 AI 이미지 제너레이터

1. 프로젝트 개요
프로젝트명: Brand-Sync AI Image Generator (가칭)
목표: 로컬 환경의 특정 디렉토리에 저장된 캐릭터 원본 이미지와 브랜드 지정 색상을 기반으로, 고정 프롬프트와 변동 프롬프트를 조합하여 형태가 무너지지 않는 다양한 포맷의 이미지를 자동 생성한다.
주요 접근법: 텍스트 프롬프트에만 의존하지 않고 이미지 자체를 입력값으로 처리하는 **Nano Banana 기반 아키텍처(또는 Gemini API)**를 활용하여 캐릭터의 이목구비, 비율 등 아이덴티티를 보존한다.
작업자가 최소한의 변수(오브젝트, 포즈)만 입력해도, 시스템이 '고정 프롬프트'를 자동 병합하여 브랜드 톤앤매너를 강제한.

2. 주요 기능 요건 (Core Features)
2.1. 로컬 기반 에셋 및 설정 관리
디렉토리 모니터링: 시스템은 지정된 로컬 폴더(예: /assets/character/)에 있는 참조용 캐릭터 이미지(최대 14장 동시 참조 가능 수준)를 모델 생성의 직접적인 입력값으로 로드한다.
글로벌 환경 변수 (Config): config.json 또는 .env 파일을 통해 브랜드의 주색상(Main Color)과 보조색상(Sub Color)의 Hex 코드를 관리하고 프롬프트 변수로 주입한다.
출력 포맷 설정: 생성될 이미지의 포맷(PNG, JPG), 해상도(비율), 투명 배경 여부(Remove BG) 등을 로컬 설정값으로 제어한다.

2.2. 프롬프트 시스템 (Prompt Engine)
유동(변동) 프롬프트 입력: 작업자는 CLI 또는 로컬 GUI를 통해 변경하고자 하는 핵심 키워드(예: "선물 상자를 든 모습", "계산기")만 간단하게 입력한다.
LLM 기반 Prompt Enhancer (선택/확장): 작업자가 입력한 단순 키워드를 내부적으로 최적화된 영문 프롬프트로 확장 번역하는 전처리 과정을 거친다. (영어 프롬프트가 포즈/상황 묘사 반영에 더 유리함)
고정 프롬프트(Fixed Prompt) 자동 맵핑: 사용자의 유동 프롬프트 앞뒤에 사전에 정의된 브랜드 고정 수식어를 자동으로 결합한다.
결합 구조 예시: {고정 프롬프트: 렌더링 스타일} + {유동 프롬프트: 작업자 입력 포즈/오브젝트} + {고정 프롬프트: 컬러 변수 적용 및 조명 환경}.

2.3. AI 모델 연동 (API Integration)
이미지 최적화 모델 호출: 캐릭터 형태 보존에 뛰어난 Nano Banana 엔진(또는 기업용 보안이 적용된 Gemini 3 Pro Image API 등)에 파라미터를 전달하여 이미지를 생성한다.
보안 및 데이터 보호: 사내 캐릭터 자산 유출 및 AI 학습 데이터 활용을 방지하기 위해 로컬 환경에서 실행하며, 필요시 엄격한 보안 정책이 적용된 기업용 API(Enterprise API)로 통신한다.

3. 프롬프트 구조화 상세 스펙
변동(Variable) 변수: [OBJECT], [POSE], [BACKGROUND]
고정(Fixed) 변수: [MAIN_COLOR], [SUB_COLOR], [CHARACTER_IMAGE_PATH]

4. 시스템 아키텍처 및 워크플로우
Init: Python 스크립트 실행 시 로컬 디렉토리에서 Character.png 및 config.json (색상값) 로.
Input: 사용자로부터 생성할 상황(변동 프롬프트) 텍스트 입력 수.
Enhance & Merge: 입력된 텍스트를 LLM을 통해 정교한 영어 문장으로 다듬은 뒤, 시스템의 고정 프롬프트 문자열과 결합.
Generate: 결합된 텍스트 프롬프트와 참조 캐릭터 이미지 원본을 함께 이미지 생성 AI(API)로 전송.
Post-Process & Save: 반환된 결과물을 설정된 해상도/포맷으로 변환(필요시 업스케일링 및 배경 제거)하여 /output/ 디렉토리에 타임스탬프와 함께 자동 저장.

5. 기대 효과 및 해결 과제
형태 안정성 확보: 참조 이미지를 직접 입력값으로 사용함으로써 원본 캐릭터의 비율이나 이목구비가 틀어지는 현상 방지.
작업자 피로도 감소: 텍스트/색상 등의 고정값들이 자동 수식어로 결합되므로, 매번 긴 프롬프트를 짤 필요 없이 작업자의 단순 키워드만으로 고품질 결과물 도출.
일관된 자산 구축: 사전에 정의된 표준 가이드라인 안에서만 이미지가 생성되므로 브랜드 톤앤매너를 강제할 수 있음.
